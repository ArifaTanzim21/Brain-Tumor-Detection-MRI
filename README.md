# Brain-Tumor-Detection-MRI

A comparative study of **traditional image processing techniques** and **deep learning-based approaches** for automated brain tumor detection in MRI scans.

## Overview

This project explores two approaches for detecting brain tumors from MRI images: a traditional rule-based image processing pipeline and a Convolutional Neural Network (CNN). The study compares their performance, computational requirements, and interpretability for medical image analysis.

## Methodology

* MRI image preprocessing and enhancement
* Traditional image processing using CLAHE and median filtering
* Tumor segmentation using adaptive thresholding and morphological operations
* Feature extraction using area, circularity, and solidity
* Rule-based tumor classification
* CNN-based deep learning classification
* Performance evaluation using accuracy, sensitivity, specificity, and F1-score

## Results

The project demonstrated the differences between traditional image processing and deep learning approaches in brain tumor detection. The traditional method offered faster processing and better interpretability, while the CNN approach provided strong performance in identifying complex tumor patterns.

* **Traditional Method:** Rule-based classification with handcrafted features
* **Deep Learning Method:** CNN-based automatic feature learning
* **Best CNN Accuracy:** 97.33%
* **Processing Speed:** Approximately 1 second/image using GPU

## Technologies

**Python/MATLAB | Image Processing | Deep Learning | CNN | Medical Imaging | MRI**

## Key Takeaway

The study highlights the potential of combining traditional image processing for rapid initial screening with deep learning for more precise confirmation, creating a possible hybrid framework for automated brain tumor detection.
